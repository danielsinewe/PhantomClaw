# Changelog

## 0.1.0

- Introduced the first canonical PhantomClaw automation bundle.
- Added Peerlist follow/unfollow metadata, schema, examples, and README.
- Documented `peerlist_profile_followers` as the north-star metric.
- Set conservative defaults: 3 verified follows per day and 1 verified follow per run.
- Required durable daily cap enforcement through `automation_action_events_v1`.
