FAQ

Questions before you hand work to an agent.

Will it run too aggressively?

No. You set action limits, launch cadence, protection rules, and stop conditions before a workflow runs.

Can I customize the workflow?

Yes. Workflows are built as editable playbooks with clear settings for action type, caps, cadence, and reporting.

What should I measure first?

Start with one north-star metric. For Peerlist, track your profile followers daily and scale only when it improves.
