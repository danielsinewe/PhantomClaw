# Code of Conduct

## Our Standard

We want this project to be useful, direct, and safe for contributors of different backgrounds and experience levels.

Examples of behavior that contribute to a positive environment include:

- giving actionable, respectful feedback
- assuming good intent while still being technically rigorous
- documenting decisions clearly
- helping newcomers understand repo conventions

Examples of unacceptable behavior include:

- harassment, intimidation, or discrimination
- publishing private information without permission
- hostile or bad-faith review behavior
- spam, trolling, or repeated off-topic disruption

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct and may remove, edit, or reject comments, commits, issues, or other contributions that violate it.

## Scope

This code of conduct applies in project spaces and in public spaces when someone is representing the project.

## Reporting

Report unacceptable behavior privately to the maintainers.
